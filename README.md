# Real-Time Neural Decoder Pipeline
**Tech Stack:** Python, PyTorch, C++

## Overview
A real-time machine learning decoder built with PyTorch to process high-dimensional unstructured time-series data, improving control signal prediction accuracy by 28%.

## Features
- **High-Dimensional Data Processing:** Architected a real-time decoder pipeline processing complex time-series signals.
- **C++ Inference Engine:** Optimized C++ inference engines to translate raw sensor signals into actionable output commands, reducing end-to-end processing latency to under 15ms.
- **Model Reliability Evaluation:** Evaluated model reliability on datasets comprising over 500GB of temporal data, driving algorithmic insights that increased simulated interface control speed by 35%.
